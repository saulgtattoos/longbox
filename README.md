# Robert's Vault

Comic book inventory management system built for collectors who are serious about their collection.

## Stack
- React + Vite
- Tailwind CSS v4
- Supabase
- Claude AI (Anthropic)
- Deployed on Vercel

## Local Development
```bash
npm install
npm run dev
```
