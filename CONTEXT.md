# Robert's Vault Context

## Project
Comic book inventory app for a private collector. Goal is to catalog a large collection quickly using AI assisted entry, then export clean data for selling.

- red: #c0392b

## Fonts
- Syne (headings, wordmark)
- Inter (body)
- JetBrains Mono (labels, mono)

## Design Direction
Comic book aesthetic. Bold, dark, ink textures. Gold accents throughout.

## Data
- localStorage for now
- Supabase migration planned
- Key: longbox_comics

## Rules
- No hyphens or dashes in generated copy
- All labels in JetBrains Mono
- Gold borders on header and footer
